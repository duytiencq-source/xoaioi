📄 Xuất index.html

Tất cả ảnh đã chỉnh sửa sẽ được nhúng vào file HTML duy nhất. Tải về và thay thế file cũ trên GitHub.

🚀 Hướng dẫn deploy GitHub Pages

Sau khi xuất file index.html, làm theo các bước sau để cập nhật website:

1. Vào repo GitHub của bạn
2. Bấm vào file index.html hiện tại
3. Bấm nút ✏️ Edit (bút chì)
4. Xóa toàn bộ nội dung cũ
5. Dán nội dung file mới vào
6. Bấm "Commit changes"
7. Website tự cập nhật sau ~1 phút

Hoặc dùng nút "Upload files" để thay thế trực tiếp.
